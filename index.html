Portfolio

Some recent projects

  • A Bitter Parent

    This app aims to provide parents with a simple tool to improve the eating habits of their children. Utilizing the Nutritionix API, a food item can be queried and if determined to contain a high amount of sugar, a healthier alternative will be suggested. Each pair can then be saved in a Firebase library for future review.

    Try it Live View the Code

  • The Phishing Net

    Is there nothing we can do to protect ourselves from phishing scams? Using this app, users can quickly and safely check the phishing history of queried websites as reported by the PhishStats API. The previous searches are saved to a Firebase library for users to review.

    Try it Live View the Code

  • That Actor From That Thing

    What do you do when the name of an actor is on the tip of your tongue? By narrowing down a film by rating, genre, and decade, this app will connect with the TMDB API and offer targeted suggestions to the users that will hopefully shed some light on the thespian in question.
